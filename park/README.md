# 🌌 ParkEase — The Aurora Era
> **Advanced IoT Parking Solutions | Engineered with Bayesian Precision**

ParkEase is a next-generation urban mobility platform designed to eliminate parking stress through real-time slot tracking, high-end "Aurora" aesthetics, and seamless UPI integration.

## 🛠️ Technology Stack
- **Frontend:** Vibrant Aurora UI (HTML5, Vanilla CSS3, Javascript ES6+)
- **Backend:** Python Flask API
- **Database:** SQLite3 with JSON Transaction Logging
- **PDF Engine:** FPDF with Aurora Branding
- **Payment:** UPI Deep-Link Simulation

## 🚀 Quick Setup
1. **Clone the repository:**
   ```bash
   git clone https://github.com/Deepak230107/parkingslot.git
   ```
2. **Install dependencies (Python 3.10+):**
   ```bash
   pip install flask flask-cors fpdf
   ```
3. **Run the backend:**
   ```bash
   py backend/app.py
   ```
4. **Launch the platform:**
   Open `index.html` in any modern browser.

## 📊 Administrative Suite
